#### 3. Albany Engineered Composites (AEC)
**Type:** 3D woven composite preforms, aerospace structural components (LEAP engine fan blades, F-35 structures)
**Site:** Rochester, NH (HQ + R&T Center) + Salt Lake City, UT
**Relevance:** VERY HIGH. Interface failure research directly informs how fiber-to-matrix load transfer works in these structures.

**Internship status:**
- Multiple 2026 intern roles confirmed open

**Apply:** https://www.albint.com/careers/internships/ — directs to ADP recruiting portal

**Key connection:** Through-thickness failure in 3D woven composites is fundamentally a fiber-matrix interface problem. Candidates who can connect their interface failure mechanics research to how woven preform architectures distribute load to the matrix have an extremely credible research pitch for AEC.

## Notes on Application Framing

**For composites material manufacturers (Hexcel, Toray, Solvay, AEC):**
Lead with the interface failure research → "The failure mechanism I study computationally — how materials fail at boundaries — is the same physics that governs delamination in your prepreg products. [My prior DoD lab work] translated molecular-scale results into processing windows; I want to see how those windows translate to actual production."

**For aerospace structures (Spirit, Ducommun, Triumph):**
Lead with the manufacturing guidance output → "[My prior DoD lab work] produced explicit processing windows from atomic-scale simulations — the kind of result that directly constrains what happens on the production floor. I want to be on that floor and see the connection firsthand."

**For advanced manufacturing startups (Machina, Divergent):**
Lead with computational + ML angle → "I build simulation workflows that output process guidance. [At my prior DoD lab], that was MD for [your material system]. The underlying challenge — using computation to define optimal process parameters — is the same challenge your team solves for [forming/additive]."
