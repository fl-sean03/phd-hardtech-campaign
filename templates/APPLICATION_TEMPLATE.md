## Application Checklist

Complete these steps in order. Check each box when done.

- [ ] **Read STATUS.md** for this company. Confirm score, fit rationale, and flags before proceeding.
- [ ] **Verify posting is live.** Navigate to the application URL. Confirm you can click "Apply."
      If the URL returns a 404 or "position filled," stop and note in STATUS.md.
- [ ] **Create account or log in** to the company's ATS (Workday, Greenhouse, Lever, iCIMS, etc.)
      Use your primary email from POSITIONING_TEMPLATE.md Section 1.
- [ ] **Fill basic profile.** Name, email, phone, address, LinkedIn, GitHub, degree, graduation date.
      Use Form Fields table above — copy exactly.
- [ ] **Upload resume.** Use the file specified in Resume Selection section above.
      Name the file: `[LastName]_[FirstName]_Resume_[CompanyShortname].pdf` if the ATS allows.
- [ ] **Upload transcript.** Unofficial is almost always acceptable unless posting specifies official.
      Have PDF ready. File naming: `[LastName]_[FirstName]_Transcript_Unofficial.pdf`
- [ ] **Paste cover letter.** If the ATS has a cover letter field, paste from Cover Letter section
      above. If it only accepts file upload, save the cover letter as a PDF first.
      Name: `[LastName]_[FirstName]_CoverLetter_[CompanyShortname].pdf`
- [ ] **Answer screening questions.** Use responses from Screening Questions section above.
      Do not leave any field blank — if a question does not apply, write "N/A" rather than
      leaving it empty.
- [ ] **Answer any additional role-specific questions** not covered in this template.
- [ ] **Review the full application before submitting.** Check: correct resume uploaded,
      no placeholder text remaining, email is correct, graduation date is correct.
- [ ] **Submit.**
- [ ] **Record submission.** Update the Submitted field at the top of this document with the date
      and time. Update STATUS.md status to 🟢 Submitted.
- [ ] **Log in tracker.** Add a row to the campaign tracker spreadsheet or tracker doc with:
      company, role, date submitted, deadline, follow-up date (2 weeks after submission).
- [ ] **Set follow-up reminder.** If no response in 2 weeks, send a brief follow-up to the
      recruiter or hiring manager using OUTREACH_TEMPLATE.md follow-up section.
